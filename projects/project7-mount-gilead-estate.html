Mount Gilead Estate

branding / web
Challenge

Mount Gilead Estate is a high-class retirement community for over 55s. Key areas of importance included tablet and mobile optimisation, intuitive user controls and high visual impact through use of product showcase images.

Solution

The key to creating a website equally at home on everything from a mobile device to desktop was through the use of a modern and responsive web framework. Zurb's Foundation framework was utilised to create flowing layouts with clean dynamic functionality.

Result

The flowing and responsive nature of the website give the user an interactive experience of Mount Gilead Estate simply not possible with traditional print-based media.
